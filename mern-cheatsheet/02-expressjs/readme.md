## Quick Start

```bash
# Install dependencies
npm install

# Serve on localhost:5000
npm run dev
```
## Scripts & Versions
node -v > @11.6.0
npm init -y
npm i express@4.16.4
npm i -D (--save-dev) nodemon@1.18.9
npm i moment@2.24.0
npm i uuid@3.3.2
npm i express-handlebars@3.0.1
